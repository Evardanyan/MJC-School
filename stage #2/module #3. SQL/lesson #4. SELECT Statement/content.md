# SELECT statement

The **SELECT** statement is used to retrieve data from the database.

## Syntax

```
SELECT column1, column2, ...
FROM table_name;
```

If you want to select all the fields available in the table, use the following syntax:

```
SELECT * FROM table_name;
```
